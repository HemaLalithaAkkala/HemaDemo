git init

       initializes a new git repository and bigins tracking an existing directory.
git clone

       creates a local copy of a project that already exists remotely.
git add

      stages a change.
git commit

     saves the snapshot to the project history and completes the change tracking process.
git status

    shows the status of changes.which are modified or stages.
git branch

    shows the branches being worked locally.

git merge

    merges lines of development together.
git pull

   updates the local line of development with updates from its remote part.
git push

  updates the remote repository with any commits made locally to a branch
